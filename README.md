# French Poem Generator 🇫🇷✨

An interactive web application that generates creative French poems based on user input. This project explores natural language generation using JavaScript, APIs, and front-end web development techniques.

## Overview

The French Poem Generator allows users to input a topic, word, or idea and instantly receive a uniquely generated poem in French. The project demonstrates how user input can be transformed into creative text using modern web technologies.
